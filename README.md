SCAS — Sistema de Controle de Atendimentos

Sobre o projeto
O **SCAS** é um sistema desenvolvido em **Python (Tkinter) com Banco de Dados SQLite** em **2020**, criado para auxiliar no registro e acompanhamento de **atendimentos dos sistemas Fiorilli Software**.

Ele foi projetado para uso interno, oferecendo uma interface simples e intuitiva para anotar chamados, observações e detalhes de atendimentos realizados.

---

Funcionalidades principais
- Cadastro e consulta de atendimentos  
- Banco de dados local integrado  
- Interface gráfica com **Tkinter**  
- Sistema leve e rápido  
- Executável (.exe) pronto para uso, com todas as dependências incluídas  

---

Estrutura do projeto
A pasta do sistema contém:
- `SCAS.exe` — executável principal do sistema  
- `DLLs/` — bibliotecas necessárias para funcionamento  
- `Database/` — arquivos do banco de dados local  
- `imagens/` — capturas de tela e ícones do sistema

---

Como executar:
1. Extraia todos os arquivos no (C:\SCAS)
2. Dê um duplo clique em `SCAS.exe`.  
3. O sistema abrirá automaticamente.

> ⚠️ **Importante:**  
> Não altere a estrutura das pastas nem mova o executável fora da pasta `C:\SCAS`, pois o sistema utiliza caminhos relativos para acessar o banco de dados e imagens.
